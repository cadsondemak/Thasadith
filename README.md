# Srisakdi
Original designed by Prof. Srisakdi Charmonman Ph.D. and commissioned to Akarameytee aka Pairoj Peamjakapong and Bavorn Joradol to finish the drawing for SIPA Thailandâ€™s National fonts project. Srisakdi is an old fashion handwriting based on early Thai Ratanakosin style of controlled handwriting using western ink and paper. The old fashion handwriting contributes the sense of old world document text face. An ideal of old style long reading Thai text face for retrospective occasions.
